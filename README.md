# clima-api

> Proyecto hecho con:

* Java 8.
* Jetty.
* Maven.
* MongoDB _mLab_
* Spring Boot.
* JSON _org.json_
* Math _Apache Commons_

* Obtener todas las consultas previas de clima:

   http://...:9000/api/

* Nueva consulta:

   http://...:9000/api/clima?dias={dias} => http://...:9000/api/clima?dias=270
   
* Ver los planetas cargados en la base de datos:
   
   http://...:9000/api/planetas/
